# AI Paraphrase Tool

## Overview
The **AI Paraphrase Tool** is a Python-based application that uses AI models to automatically rephrase or paraphrase text. It helps users generate alternative versions of sentences while maintaining the original meaning, making it useful for writing, content creation, and language learning.

## Features
- Paraphrases sentences or paragraphs using AI.
- Preserves the original meaning while changing the structure and wording.
- Provides grammar-corrected outputs.
- Scores paraphrases using evaluation metrics like BLEU and ROUGE (optional).
